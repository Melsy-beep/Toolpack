# Toolpack
Набор скриптов для упрощения жизни в Linux
Toolpack - это колеекция утилит, упрощающих повседневную работу в Linux.
Цель - сделать систему удобной, прозрачной и гибкой под себя
# Использование
Запуск скриптов:
bash net_cheker.sh
python3 file_sorter.py
